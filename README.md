# Adam's Rib 🍖

Welcome to the repository for **Adam's Rib**, a premium web application built with **Laravel 12** and **Livewire**.

This project is a labor of love—a digital tribute and a "markup" prototype designed to encourage a dear friend as he considers sharing his extraordinary culinary skills with the community. Named **Adam's Rib**, the site honors the biblical story of Eve being created from Adam's rib, reflecting the deep, foundational connection and the "sweet loving energy" that comes from supporting those we care about. In many circles, calling someone your "rib" is the ultimate expression of partnership and devotion; this platform is built with that same spirit of support.

While the underlying logic is robust enough to handle complex inventory (like the automobile project I previously experimented with), the heart of *this* application is purely focused on the art of **Bar-B-Que**.

## 🛠 The Tech Stack<



This project serves as a showcase for the **TALL stack**, providing a modern, reactive experience that feels as smooth as a slow-cooked brisket.

| Layer | Technology |
| --- | --- |
| **Language** | **PHP 8.2+** — The powerful backbone of the application. |
| **Framework** | **Laravel 12** — The latest evolution of the elegant PHP framework. |
| **Frontend Framework** | **Laravel Livewire 4** — Real-time reactivity for menus and orders without writing JS. |
| **UI Components** | **Flux UI** — Beautiful, pre-built components for a professional finish. |
| **Styling** | **Tailwind CSS v4** — Utility-first CSS for a custom, "home-cooked" design feel. |
| **Interactivity** | **Alpine.js** — Handling the lightweight client-side "sprinkles." |
| **Local Environment** | **Laravel Herd** — The high-performance development environment powering this build. |
| **Database** | **SQLite** (or MySQL/Postgres) — Storing the recipes, orders, and gallery data. |
| **Build Tool** | **Vite** — For lightning-fast asset compilation. |

## 🔥 Project Features

**Adam's Rib** is designed to capture the essence of a high-end BBQ smokehouse. Key features include:

* **The Pit Master’s Gallery:** A dynamic photo system allowing for up to **10 high-resolution images** of the food (with at least 1 required), so the texture of the bark and the smoke ring can truly shine.
* **Reactive Menu Management:** A Livewire-powered menu where categories (Ribs, Brisket, Sides) can be browsed instantly.
* **Loving Energy UI:** A clean, warm interface styled with **Tailwind CSS** that reflects the "sweet loving energy" behind the brand.
* **Robust Backend:** Leveraging Laravel 12’s refined routing and models to ensure that every "order" or "reservation" is handled with precision.

## 🏗 Development Philosophy

As I develop this in my **Herd** environment, I am focusing on the sacred trio of web development: **Routes, Controllers, and Models**.

In the world of **Adam's Rib**, the **Model** represents the food and the soul, the **Route** is the path to the table, and the **Livewire Component** acts as the server, bringing the heat directly to the guest. By mastering these connections, I’m building a platform that is as reliable as a well-seasoned offset smoker.

## 🚀 Getting Started

To view the demo in your own local environment:

1. **Clone the repository:**
`git clone https://github.com/your-username/adams-rib.git`
2. **Install PHP & JS dependencies:**
`composer install && npm install`
3. **Configure your environment:**
`cp .env.example .env`
4. **Prepare the pit (Database):**
`php artisan migrate --seed`
5. **Serve the site:**
`npm run dev` (and access via your Herd `.test` domain).
